# hamroSewa

A Local On-Demand Service Platform for Nepal — connecting customers with local service providers in real time.

Major Project — Bachelor of Computer Application, Nepal College of Information Technology, Pokhara University.

---

## Repository Structure

```
hamroSewa/
├── frontend/       # React Native (Expo) — mobile app
├── backend/        # Express.js — REST API
├── admin_panel/    # Next.js — admin dashboard
└── README.md
```

---

## Setup

### Frontend (React Native + Expo)

```bash
cd frontend
npm install
npx expo start
```

### Backend (Express.js)

```bash
cd backend
npm install
npm run dev
```

### Admin Panel (Next.js)

```bash
cd admin_panel
npm install
npm run dev
```
