QUICK REFERENCE FOR YOUR COMPUTER
═══════════════════════════════════════════════════════════════════════════════

1. WHAT YOU NEED ON YOUR COMPUTER
   ─────────────────────────────────

   ✓ Node.js v18 or newer        → Download: https://nodejs.org
   ✓ PostgreSQL v15 or newer     → Download: https://www.postgresql.org/download
   ✓ Git                          → Download: https://git-scm.com
   ✓ Code Editor (VS Code)        → Download: https://code.visualstudio.com
   ✓ Terminal/Command Prompt      → Already on your computer


2. VERIFY YOUR INSTALLATION
   ────────────────────────────

   Open Terminal/Command Prompt and run these commands to verify:

   node --version          # Should show v18 or higher
   npm --version           # Should show v9 or higher
   psql --version          # Should show PostgreSQL 15 or higher
   git --version           # Should show git version


3. ONE-MINUTE SETUP
   ─────────────────

   For macOS/Linux:
   ─────────────────
   1. Open Terminal
   2. Navigate to project: cd /path/to/hamroSewa
   3. Run: chmod +x setup.sh && ./setup.sh
   4. Follow the prompts

   For Windows:
   ────────────
   1. Open Command Prompt (cmd.exe)
   2. Navigate to project: cd C:\path\to\hamroSewa
   3. Run: setup.bat
   4. Follow the prompts


4. MANUAL SETUP (If preferred)
   ───────────────────────────

   Step 1: Install Dependencies
   ─────────────────────────────
   Terminal 1:
   $ cd backend
   $ npm install
   $ cd ../frontend
   $ npm install

   Step 2: Configure Backend
   ─────────────────────────
   $ cd backend
   
   Create file: .env.local
   
   Content:
   DATABASE_URL="postgresql://postgres:password@localhost:5432/hamrosewa"
   JWT_SECRET="your-secret-key-here"
   PORT=5000
   NODE_ENV="development"
   CORS_ORIGIN="http://localhost:3000,http://localhost:19000"
   
   Then run:
   $ npx prisma migrate dev --name init
   $ npx prisma generate

   Step 3: Configure Frontend
   ──────────────────────────
   $ cd ../frontend
   
   Create file: .env.development.local
   
   Content:
   EXPO_PUBLIC_API_URL="http://localhost:5000/api"
   EXPO_PUBLIC_API_TIMEOUT="10000"
   EXPO_PUBLIC_DEBUG_MODE="true"


5. RUNNING THE PROJECT (3 TERMINALS)
   ─────────────────────────────────

   Terminal 1 - Database (Optional but helpful):
   $ cd backend
   $ npx prisma studio
   
   This opens: http://localhost:5555 (database viewer)

   Terminal 2 - Backend Server:
   $ cd backend
   $ npm run dev
   
   Wait for: "[server] listening on http://localhost:5000"

   Terminal 3 - Frontend App:
   $ cd frontend
   $ npm start
   
   Choose:
   - Press 'w' for Web Browser (easiest)
   - Press 'a' for Android Emulator
   - Press 'i' for iOS Simulator


6. VERIFY EVERYTHING WORKS
   ─────────────────────────

   ✓ Open http://localhost:5000/api/health
     Should see: {"status":"ok"}

   ✓ Open http://localhost:19000
     Should see: hamroSewa app loading

   ✓ Go to http://localhost:5555
     Should see: Database browser interface


7. TROUBLESHOOTING YOUR SETUP
   ──────────────────────────

   Port 5000 already in use?
   $ PORT=5001 npm run dev

   PostgreSQL won't connect?
   $ psql -U postgres
   (Verify connection works, then exit)

   Dependencies missing?
   $ rm -rf node_modules package-lock.json
   $ npm cache clean --force
   $ npm install

   Can't find commands?
   $ npm install -g expo-cli
   $ npm install -g prisma

10. COMMON COMMANDS DURING DEVELOPMENT
    ────────────────────────────────────

    Backend:
    npm run dev              Start dev server with auto-reload
    npm run build           Build for production
    npm run lint            Check code style
    
    Frontend:
    npm start               Start Expo dev server
    npm run web            Start web version
    npm run lint            Check code style
    
    Database:
    npx prisma studio      Open database GUI
    npx prisma generate    Update Prisma client
    npx prisma db reset    Reset database (⚠️ DELETE ALL DATA)
