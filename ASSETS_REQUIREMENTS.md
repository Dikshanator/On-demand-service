# Professional Assets Required for hamroSewa App

## Executive Summary
This document outlines all professional assets needed to transform the hamroSewa on-demand service platform from a prototype using placeholder icons into a polished, production-ready application.

---

## 1. BRANDING & LOGO

### App Logo & Icon
- **App Logo** - 512x512 PNG + SVG

  - Color variants: Primary color version
  - Files needed: Icon for iOS, Android, and web

- **Logo (Dark Mode)** - 512x512 PNG + SVG
  - Alternative logo optimized for dark theme
  - Maintains brand recognition in low-light scenarios

### Favicon
- **Website Favicon** - 16x16, 32x32, 64x64 PNG
- **Apple Touch Icon** - 180x180 PNG

## 12. ACCESSIBILITY & INTERNATIONALIZATION

### Accessibility Requirements
- **High Contrast Versions** - All icons
  - Enhanced contrast for accessibility
  - 4.5:1 minimum contrast ratio for text
  
- **Monochrome Versions** - All status icons
  - For color-blind users
  - Use patterns + shape + color
  
- **ARIA Labels** - All icons
  - Semantic naming for screen readers
  - Descriptions for complex icons

### Dark Mode Support
- **All assets** - Dark theme variants
  - Inverted colors for dark backgrounds
  - Proper contrast in both light and dark modes

## 13. FILE SPECIFICATIONS

### General Requirements
- **Formats**: PNG (raster), SVG (vector), Lottie JSON (animations)
- **PNG Specifications**:
  - Resolution: 1x, 2x (for retina), 3x (for ultra-high DPI)
  - Transparent background (PNG-32)
  - Optimized for web (<100KB for 512px assets)
  
- **SVG Specifications**:
  - Scalable up to 2x original size minimum
  - Path-based (not text)
  - Properly minified
  - Color values defined for easy recoloring
  
- **Lottie JSON**:
  - Loop duration: 2-3 seconds (preferred)
  - Performance optimized
  - Exported from After Effects or Figma
  - File size <200KB
